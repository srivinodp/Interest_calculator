## ‚ú® Features
*   **Dynamic Calculation**: Interest is computed automatically in real-time as you type or slide. No "Calculate" button needed.
*   **Interactive UI**: Beautiful Material Sliders for principal amount and interest rate adjustments.
*   **Date Specific**: Calculates exact interest based on custom Start and End dates (DD/MM/YYYY).
*   **Two Modes**: Simply toggle between **Tirageta** (compound variant) and **Plain** (simple) interest models.
*   **Compact & Lightweight**: Designed to be easy on battery and fast on older devices (Min SDK 22).

## üöÄ Getting Started
1. Clone this repository and open the project in **Android Studio**.
